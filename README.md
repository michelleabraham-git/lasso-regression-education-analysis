# Lasso-regression-education-analysis
Predicting Student–Teacher Ratios Using LASSO Regression

Research Question: 
What district-level development indicators explain variation in student–teacher ratios in Indian primary schools?

LASSO Regression Analysis

This project explores the determinants of student–teacher ratios in Indian primary schools using LASSO regression.

Methods used:
LASSO regression
Comparison with OLS

Tools:
R
The analysis demonstrates how regularization methods can identify relevant predictors in high-dimensional datasets.
